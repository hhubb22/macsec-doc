# Annex D (Normative): PICS Proforma for EDEs

- Extends Annex A with specific conformance statements for Ethernet Data Encryption (EDE) devices (Clause 15).
- Captures mandatory/optional behavior for EDE-M, EDE-CS, EDE-CC, and EDE-SS variants.
- Includes sections for topology support, EAPOL address usage, VLAN handling, replay protection, and interoperability constraints.
- Detailed tables will be reformatted into Markdown checklists in a future iteration.
