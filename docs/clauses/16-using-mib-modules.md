# Clause 16: Managing EDEs with MIB Modules

- Recommends how SNMP managers should use the SecY MIB (Clause 13) to operate EDE variants (Clause 15).
- Highlights security considerations when exposing EDE-specific management objects, including confidentiality of key material and PN status.
- Advises alignment with provider/customer operational policies when configuring replay windows, traffic classes, and group-address filtering.
